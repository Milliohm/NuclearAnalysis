# NuclearAnalysis
Programm zur Analyse von Unternehmen, die im Bereich SMRs, nuclear reactors generation IV und Fusionsenergie tätig sind

Ziel: 
Ein Programm, idealerweise im Web verfügbar, dass mir die neuesten Nachrichten zu der Entwicklung neuer Reaktoren liefert. Das Programm soll mir Basiswissen für die Investition in diese Unternehmen liefern.
